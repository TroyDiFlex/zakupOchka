<h1 align="center">zakupOchka</h1>

<p align="center">
  Скрипт-виджет Tampermonkey для поиска игр в Steam по проценту скидки и/или цене в заданном диапазоне.
</p>

<hr>

<p>
Мне лень писать здесь что-то ещё, этот текст всё равно увидят максимум полтора землекопа.
Но если вдруг кто-то всё-таки решит это устанавливать, просто создайте новый файл в Tampermonkey,
и вставьте туда код.
</p>
