<h1 align="center">ЗакупОчка</h1>

<p align="center">
  Скрипт-виджет Tampermonkey для поиска игр в Steam по проценту скидки и/или цене в заданном диапазоне.
</p>

<hr>

- Два режима отображения:
  - по умолчанию со страницы исчезают все игры, не попадающие в заданный диапазон.
  - без скрытия подсвечиваются игры, подходящие под условия поика; неподходящие остаются видимыми.
  
- Скрипт сканирует текущую страницу Steam, собирает данные о цене и проценте скидки и применяет фильтры и сортировку в реальном времени.
